# 输出评估方法

## 评估维度

每次对 skill 做重大修改后，用以下维度评估输出质量：

| 维度 | 通过标准 | 检查方式 |
|------|---------|---------|
| 时间线合规 | 无未来时间、无重叠、无空窗、每段≥7个月 | 人工抽查 5 份简历 |
| 项目制重构 | 无岗位职责罗列，全部改写为项目 | 人工抽查 |
| 数据闭环 | 个人优势数字 = 项目数字严格加总 | 反向核对 |
| 数据质检 | 年产≤600、留资数据合理、无团队累计算个人 | 人工 + 脚本校验 |
| 跨行业迁移 | 非新媒体经历至少产出 3 个新媒体项目 | 专门测试 3 份跨行简历 |
| docx 导出 | 两个 docx 正常生成、无格式错乱 | 自动测试 |
| AI 标注 | 所有 AI 补充/推断/迁移都有标注 | 自动检查标记 |

|------|------|------|------|
